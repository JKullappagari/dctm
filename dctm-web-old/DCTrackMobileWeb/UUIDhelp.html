Windows Mobile devices

Only Zebra/Motorola devices running Windows Mobile/Windows CE are supported.

Steps to get UUID:

1. Click
2. Click "Settings".
3. Click "System" (In case Win Mobile 5: Page tab / In case of Win Mobile 6.0 or later: Icon).
4. Click "System Info".
5. Combine UUID part 1 and Part 2 to get UUID (Ex: UUID part 1 + UUID part 2).
6. Combined ID will be of 32 characters in length.

Android devices

Zebra devices are primary target devices, but application will work with standard Android based phones also with barcode reading via image scanner.

Steps to get DeviceId:

1. Install DCTrackMobile application on the device.
2. Go to Login screen of DCTrackMobile.
3. Make note of DeviceId shown in Login screen and use it as device id to register android device. Generally DeviceId will be 8-24 characters in length.
