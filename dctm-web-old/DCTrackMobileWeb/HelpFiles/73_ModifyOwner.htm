Modify Owner

 

To modify an Owner,

5.      Navigate Masters>>Owner on the DCTrackMobileWeb menu bar.

6.      On the Owner screen, click the Edit  icon corresponding to the Owner you want to modify.

Figure 92: Owner Screen – Edit details

7.      Edit the details.

8.      Click Save  button to update the database with the changes or click Reset button to refresh page fields.

Upon saving, you will see the successful update message.
