Modify Group

 

To modify a Group,

1. Navigate User Administration>> Group on the DCTrackMobileWeb menu bar.

2. On the Group screen, click the Edit  icon corresponding to the User Group you want to modify.

You will see the existing values displayed on the screen.

Figure 165: Group Screen – Edit details

3. Edit the details to be modified.

4. Click Save  button to update the database with the changes or click Reset button to refresh page fields.

Upon saving, you will see the successful update message.

 

Figure 166: Group Screen – Successful Update
