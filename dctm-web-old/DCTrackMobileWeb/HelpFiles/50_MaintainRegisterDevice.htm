Maintain Register Device

 

 

The Register Device screen will enable privileged users to register and maintain Motorola devices which will be used to install DCTrackMobile application.

 

Device registration using web is compulsory, without registration DCTrackMobile application installation on that device will fail.

 

Can I Register different devices with the same name?

No.  Each device registration requires UUID (Unique ID defined by Motorola, each device will have a unique UUID) and device name. Device name is the logical name to identify the device and it should be unique.
