Create User

 

1.      Navigate User Administration>>Manage Users from the DCTrackMobileWeb menu bar. You will see the Manage Users screen.

 

Figure 173: Manage User screen

2.      Enter the information in the requested fields.

Is Password complexity defined?

Yes. Password must be at least 6 characters, no more than 10 characters, and must include at least one upper case letter, one lower case letter, one numeric digit and no spaces and no special characters allowed.

What is Site Restriction?

Site restriction enables admins to set restriction on users in terms of asset data access based on site. Site restriction can be No Access or Read orRead/Write. No Access means user cannot access or see assets belong to a particular site. Read allows user to see asset data but cannot modify existing assets or create new assets under that site. Read/Write give full access that mean user can modify existing and create new ones under that site.

 

3.      Define Site restriction for the user if needed.

4.      Click Generate Password button  to generate a random password based on the password complexity rules.

5.      Click Save  button to store data into the database.

Figure 174: Manage Users Screen- user creation

 

 

Figure 175: Manage Users - Successful user creation

 

 

6.      On successful user creation, either Password will be sent to the user email or shown on the screen so that the admin can make note of it. Share these details with the user and tell user to change the password on the first login using Change Password feature.

a.     Password will be sent to user if “SendMail” configuration setting is set to 1

b.     Password will be shown on the screen to admin if “SendMail” configuration setting is set to 0
