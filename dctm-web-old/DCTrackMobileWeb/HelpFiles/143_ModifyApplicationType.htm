Modify Application Type

 

To modify an Application Type,

1.      Navigate Application>>Application Type on the DCTrackMobileWeb menu bar.

2.      On the Application Type screen, click the Edit  icon corresponding to the application type you want to modify.

You will see the existing values displayed on the screen.

Figure 191: Application Type Screen – Edit details

3.      Edit the details to be modified.

4.      Click Save  button to update the database with the changes or click Reset button to refresh page fields.

Upon saving, you will see the successful update message.

 

Figure 192: Application Type Screen – Successful Update
