Modify Location

 

To modify a Location,

1.      Navigate Masters>>Location on the DCTrackMobileWeb menu bar.

2.      On the Location screen, click the Edit  icon corresponding to the Location you want to modify.

You will see the existing values displayed on the screen.

Figure 30: Location Screen – Edit details

3.      Edit the details to be modified.

4.      Click Save  button to update the database with the changes or click Reset button to refresh page fields.

Upon saving, you will see the successful update message.

 

Figure 31: Location Screen – Successful Update
