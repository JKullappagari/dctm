Password Property

 

Password must be at least 8 characters, no more than 10 characters, and must include at least one upper case letter, one lower case letter, one numeric digit and no spaces allowed.

 

If password is not defined based on above criteria than system will throw an error message as shown below:

Figure 10: Password Change Error Screen

 

 

Enter the existing password, new password and re-enter the new password. All the data has to be entered correctly otherwise the new password will not be accepted by the system.

 

Some of the password related settings are part of web.config which are listed below:

 

MaxNoOfLoginAttempts

3

Number of failed login attempts after which password will be locked.

PwdExpiryDays

90

Number of days after which password will expiry and user will be shown only Change Password page.

ExpirationWarningDays

10

Number of days before the expiry date, the warning message will be displayed regarding the password expiry date.
