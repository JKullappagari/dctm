Changing Password

 

Any individual user can change his/her password by clicking the Change Password link at the top right corner of the home page. User will be presented with the below screen to input the new password:

 

Figure 9: Password change screen

 

Enter the existing password, new password and re-enter the new password. All the data has to be entered correctly otherwise the new password will not be accepted by the system.
