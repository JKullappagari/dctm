View Application Type

 

To view an Application Type,

1.      Navigate Application>>Application Type on the DCTrackMobileWeb menu bar.

You will see the Application Type Screen with a listing of all created Application Types with corresponding descriptions.

Figure 190: Application Type Screen – View details

2.      Click page number hyperlink, if applicable to view the list in another page(s).
