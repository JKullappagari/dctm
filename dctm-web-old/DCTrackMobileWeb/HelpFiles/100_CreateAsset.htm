Create Asset

 

To create Asset,

 

1.      Navigate Asset>>Create Asset on the DCTrackMobileWeb menu bar.

You will see the Create Asset screen. 

Enter all or required information

All the fields in Create Asset are mandatory?

No. some of the fields are not applicable for media such as operating system, cpu, cpu core, cpu count, host name and technology category. All these are non-mandatory fields.

Given below are the details of the asset fields:

Custodian

Custodian of the asset.

Yes

Custodian/ Owner name can be imported during asset import, alternately   custodian can be created by clicking the magnifying glass after the custodian field and then selecting an appropriate custodian name from the list.

New Custodian/Owner details can be created using Masters >> Owner page in the web.

Created Date

Yes

No

22

Barcode/EPC

Barcode/EPC value of the asset

No

This is not a mandatory field while creating the asset but this field has to be populated for recording the asset movement in the system.

 

 

2.       Click the Save  button to complete the create asset process.
