Modify Device

 

To modify a Device,

1.      NavigateMasters>>Register Device on the DCTrackMobileWeb menu bar.

2.      On the Register Device screen, click the Edit  icon corresponding to the Register Device you want to modify.

You will see the existing values displayed on the screen.

Figure 64: Register Device Screen – Edit details

3.      Edit the details to be modified.

4.      Click Save  button to update the database with the changes or click Reset button to refresh page fields.

Upon saving, you will see the successful update message.

 

Figure 65: Register Device Screen – Successful Update
