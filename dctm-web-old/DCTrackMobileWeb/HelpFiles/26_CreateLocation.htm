Create Location

 

To create a Location,

1.      Navigate Masters>>Location on the DCTrackMobileWeb menu bar.

You will see the Location screen.

Figure 25: Location Screen

2.      Enter description of the location, select an appropriate location type. There are three types of locations that are pre-defined in the system and they are: Room, Row and Rack.

a.     Room: A physical location where assets will be placed and it may consist of rows and racks.

b.     Row: A logical location where assets will be placed and it is a subset of a room.

c.     Rack: Typically represents a rack in a data center where assets are placed and it may be a subset of a row.

3.      If the location has a parent location, click the magnifying glass and select an appropriate parent location.

  Can I select parent location for room?

 No. Parent location is not applicable for room. A room has to be associated with a site using the Assignment -> Site Location Assignment option.

 

  While creating a Rack (Location) it displays all manufacturer and all models for any manufacturer – example server, storage etc. which cannot be a model for a Rack. Why?

 Currently system does not have any specific attribute for a model to distinguish. That is why it shows all the models. User has to manually choose the appropriate model that is relevant for a Rack.

 

 

4.      Specify Floor as alphanumeric string and this is applicable to Room only. This is optional field.

5.      If required specify Serial Number, Manufacturer, Model, Width, Depth, Height fields for Rack only. These fields will be enabled for a location of location type “Rack” only.

6.      Click Save  button to complete the creation process or click Reset button to refresh the page fields.

Upon saving, you will see the successful creation message and the created Location listed on the Table View.

Figure 26: Location Screen – Successful Creation
