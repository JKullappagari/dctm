Delete Application Type

 

To delete an Application Type,

1.      Navigate Application>>Application Type on the DCTrackMobileWeb menu bar.

2.      On the Application Type screen, tick on the check box corresponding to the Site to be deleted from the list.

3.      Click Delete Selected button.

You will see the confirmation message window.

Figure 193: Application Type Screen – Delete Confirmation Message

Can I select multiple Application Types for deletion?

Yes. Simply tick on the check box corresponding to the application types you would like to remove before clicking on the Delete Selected button. You can also click the Select All check box to select all items in the list for deletion.

 

4.      Click OK  button to proceed or click Cancel  button to disregard operation.

Upon deletion, you will see the successful deletion message and the deleted Application Type is no longer on the list.

Figure 194: Application Type Screen – Successful Deletion
