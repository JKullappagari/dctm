Application Map

 

To Map Application and Asset,

2.     Navigate Application>>Application Map on the DCTrackMobileWeb menu bar.

You will see the Application Map screen.

Figure 181: Application Map Screen

2. By Default all the applications will be loaded into application list data grid. Load Hosts either by typing host name in Host text box or select a location to load all the hosts which are part of that location.

3. Click Show Hosts  button to load all the host(s) based on either host text or selected location.
