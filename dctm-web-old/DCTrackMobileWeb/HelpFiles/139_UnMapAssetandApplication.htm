Un Map Asset and Application

 

To remove map between asset(s) and application(s),

1. Navigate Application>> Application Map on the DCTrackMobileWeb menu bar.

2. Load all asset(s) by either host text or location selection. Find asset, for which application(s) will be unmapped.

3. Identify the application(s) that need be unmapped. Uncheck all these application(s) from application sub list inside the selected asset. Now select the asset row.

Figure 185: Application Map – un map process

4. based on above screen shot “app1” and “app3” will be unmapped from “Host6”. Asset row need to be selected before unmapping.

5. Click Delete Map  button to delete mapping between asset and application(s). Delete map can be done on only one asset at a time.
