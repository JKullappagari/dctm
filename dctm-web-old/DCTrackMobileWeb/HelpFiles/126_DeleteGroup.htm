Delete Group

 

To delete a Group,

1. Navigate User Administration>> Group on the DCTrackMobileWeb menu bar.

2. On the Group screen, tick on the check box corresponding to the Group to be deleted from the list.

3. Click Delete Selected  button.

You will see the confirmation message window.

Figure 167: Group Screen – Delete Confirmation Message

Can I select multiple User Group for deletion?

Yes. Simply tick on the check box corresponding to the User Group you would like to remove before clicking on the Delete Selected button. You can also click the Select All check box to select all items in the list for deletion.

 

1.      Click OK  button to proceed or click Cancel  button to disregard operation.

Upon deletion, you will see the successful deletion message and the deleted User Group is no longer on the list.
