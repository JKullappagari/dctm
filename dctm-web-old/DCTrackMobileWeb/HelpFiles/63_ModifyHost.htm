Modify Host

 

To modify a Host,

1.      Navigate Masters>>Host on the DCTrackMobileWeb menu bar.

2.      On the Host screen, click the Edit  icon corresponding to the Host you want to modify.

Figure 78: Host Screen – Edit details

3.      Edit the details.

4.      Click Save  button to update the database with the changes or click Reset button to refresh page fields.

Upon saving, you will see the successful update message.

 

Figure 79: Host Screen – Successful Update
