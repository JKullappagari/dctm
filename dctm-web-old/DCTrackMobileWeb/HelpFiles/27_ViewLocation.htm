View Location

 

To view a Location,

1.      Navigate Masters>>Location on the DCTrackMobileWeb menu bar.

You will see the Location Screen with a listing of all created Locations with corresponding descriptions.

Figure 27: Location Screen – View Location details

2.      Click page number hyperlink, if applicable to view the list in other page(s).

3.      In the grid it displays the following information for a location:

a.     Location – A short name of the location

b.     Description – Description of the location.

c.     Parent Location

d.     Location Type – either Room, Row or Rack

e.     Location Tag – the Barcode/RFID tag value that was assigned for this location. This is only valid for Rack Location Type.
