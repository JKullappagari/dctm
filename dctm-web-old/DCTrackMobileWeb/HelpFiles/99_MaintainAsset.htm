Maintain Asset

 

This section describes the maintenance of Asset. As asset can be created in one of two ways:

·       By importing the asset from an excel spreadsheet by using the Import Asset option. Typically, this option is used to create assets in bulk in the system, when one has to create hundreds of assets.

·       Creating the asset by using the Create Asset option. This is recommended to be used when one is creating fewer assets.
