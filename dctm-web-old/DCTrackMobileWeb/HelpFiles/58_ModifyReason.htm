Modify Reason

 

To modify a Reason,

1.      Navigate Masters>>Reason on the DCTrackMobileWeb menu bar.

2.      On the Reason screen, click the Edit  icon corresponding to the Reason you want to modify.

Figure 71: Decommission Screen – Edit details

3.      Edit the details.

4.      Click Save  button to update the database with the changes or click Reset button to refresh page fields.

Upon saving, you will see the successful update message.

 

Figure 72: Reason Screen – Successful Update
