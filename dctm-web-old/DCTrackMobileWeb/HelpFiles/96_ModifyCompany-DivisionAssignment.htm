Modify Company-Division Assignment

 

To modify a Company-Division Assignment,

5.      Navigate Assignment>>Company and Division assignment on the DCTrackMobileWeb menu bar.

6.      The list of current Available and Assigned divisions will be displayed.

7.      Edit the current assignment by using the control arrows to assign or de-assign a site. Refer to steps 3 to 5 in Creating Company-Division Assignment for details.

8.      Click Save  button to update the database with the changes or click Reset  button to refresh page fields.

Upon saving, you will see the successful assignment message.
