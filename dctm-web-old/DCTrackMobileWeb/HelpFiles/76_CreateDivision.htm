Create Division

 

To create a Division,

1.      Navigate Application>>Division on the DCTrackMobileWeb menu bar.

You will see the Division screen.

Figure 96: Division screen

2.      Enter information for the required field.

3.      Click Save  button to complete the creation process or else click Reset button to refresh the page fields.

Upon saving, you will see the successful creation message and the created Division listed on the Table View.

Figure 97: Division Screen – Successful Creation
