User Administration

 

This chapter describes the User Administration.  The system administrators are allowed to perform all admin activities like creating new groups, manage users, access rights, users search.

The following Site Access related functionalities can be accessed from the User Administration menu on the DCTrackMobileWeb navigation bar:

·       Group

·       Group module access rights assignment

·       Manage Users

·       User Search

·       change Password
