Maintain Host

 

The Host screen will enable privileged users to define and maintain host names which will be mapped to Assets. System allows to create hosts for any asset. User has to determine if the host information is meaningful for the asset. For example, if DCTrackMobile system is used to maintain and track media such as tapes and tapes are defined as assets, then assigning a host information with the tapes will not be meaningful. However, system does not restrict that association.

Can I create a Host of the same name?

No. Host should be unique. System will show an error message that the reason already exists.
