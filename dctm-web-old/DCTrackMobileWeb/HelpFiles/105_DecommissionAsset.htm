Decommission Asset

 

Decommission Asset page will be used to decommission an asset.  Once an asset is decommissioned then you cannot edit the asset but still can perform the following action:

·       Restrict Asset

·       Permanently Restrict (Perm Restrict) Asset

·       Write-off Asset

·       Move the Asset using the Mobile Check Out and Check In functions.

A decommissioned asset can be recommissioned using “Recommission” option in Edit Asset. This option will be shown only for decommissioned assets.

Please note the following important points:

·       Once you decommission an asset, it still stays in the same location, for example, if it is a rack mounted asset, it still stays in the same U position of the rack. Only the status of the equipment gets changed to ‘Decommission’ asset.

·       If you want to move the asset and free up the space, then you need to perform Check Out and Check in operation for the asset using the mobile. The decommissioned asset should be moved to a location called ‘DECOM Room’ – this room is created by default once you install the system.

 

1.      From Search Asset >> View Asset (by clicking on individual asset’s serial no)

2.      Click Decommission  button to decommission the asset or click Cancel  button to disregard operation.

 

Figure 131: Search Asset Screen – Decommission

3.      Specify Decommission date (date should be today or day before), reason and additional comments explaining about decommission of the asset.

4.      Click Save  button to decommission an asset or click Cancel  button to disregard operation.
