Delete Host

 

To delete a Host,

1.      Navigate Masters>>Host on the DCTrackMobileWeb menu bar.

2.      On the Host screen, tick the checkbox corresponding to the Host you want to remove.

3.      Click Delete Selected  button.

You will see the confirmation message window.

Figure 80: Host Screen – Delete Confirmation Message

 

Can I select multiple Hosts for deletion?

Yes. Simply tick on the check box corresponding to the Host you would like to remove before clicking on the Delete Selected button. You can also click the Select All check box to select all items in the list for deletion.

 

4.      Click OK  button to proceed or click Cancel  button to disregard operation.

Upon deletion, you will see the successful deletion message and the deleted Host is no longer on the list.

Figure 81: Host Screen – Successful Deletion
