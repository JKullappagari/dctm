Modify Manufacturer

 

To modify a Manufacturer,

5.      NavigateMasters>>Manufacturer on the DCTrackMobileWeb menu bar.

6.      On the Manufacturer screen, click the Edit  icon corresponding to the Manufacturer you want to modify.

You will see the existing values displayed on the screen.

Figure 44: Manufacturer Screen – Edit details

7.      Edit the details to be modified.

8.      Click Save  button to update the database with the changes or click Reset button to refresh page fields.

Upon saving, you will see the successful update message.

 

Figure 45: Manufacturer Screen – Successful Update
