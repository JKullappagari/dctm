Delete Location

 

To delete a Location,

1.      Navigate Masters>>Location on the DCTrackMobileWeb menu bar.

2.      On the Location screen, tick on the check box corresponding to the Location to be deleted from the list.

3.      Click Delete Selected  button.

You will see the confirmation message window.

Figure 32: Location Screen – Delete Confirmation Message

 

Can I select multiple Locations for deletion?

Yes. Simply tick on the check box corresponding to the Locations you would like to remove before clicking on the Delete Selected button. You can also click the Select All check box to select all items in the list for deletion.

4.      Click OK  button to proceed or click Cancel  button to disregard operation.

Upon deletion, you will see the successful deletion message and the deleted Location is no longer on the list.

 

Figure 33: Location Screen – Successful Deletion
