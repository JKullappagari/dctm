View Sub Locations and Assets

 

To view sub location and assets in a location,

 

1.      Navigate Masters>>Location on the DCTrackMobileWeb menu bar.

You will see the Location Screen with a listing of all created Locations with corresponding descriptions.

2.      Click page number hyperlink, if applicable to view the list in other page(s).

3.      Click the hyperlink of a location in the location column of the table view to see the sub locations and child assets of that location.
