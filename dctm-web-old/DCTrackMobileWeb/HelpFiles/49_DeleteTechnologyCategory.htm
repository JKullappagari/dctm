Delete Technology Category

 

To delete a Technology Category,

1.      Navigate Masters>>Technology Category on the DCTrackMobileWeb menu bar.

2.      On the Technology Category screen, tick on the check box corresponding to the Technology Category to be deleted from the list.

3.      Click Delete Selected  button.

You will see the confirmation message window.

Figure 58: Technology Category Screen – Delete Confirmation Message

Can I select multiple Technology Categories for deletion?

Yes. Simply tick on the check box corresponding to the Technology Category you would like to remove before clicking on the Delete Selected button. You can also click the Select All check box to select all items in the list for deletion.

 

4.      Click OK  button to proceed or click Cancel  button to disregard operation.

Upon deletion, you will see the successful deletion message and the deleted Technology Category is no longer on the list.
