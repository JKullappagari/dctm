Unlock & Reset Password

 

When a user account gets locked because of 3 times failed login attempt. This feature will be used by Administrator to unlock and reset password of that user.

How to reset the password only?

Use same Unlock & Reset button. This button can be used to unlock & reset password of a user account and also for reset password only.

 

1.     Click on Edit icon  of a user, whose account needs to be unlocked in User Search screen.

2.     Click Unlock & Reset button  to unlock and reset the password of the user. A random password will be generated and assigned to the user.

3.     After the successful reset,

a.     A dialog box will display the new password and administrator needs to pass this new password to user and request him to change the password on the first login in case of SendMail = 0

b.     New password will be directly sent to the user in case of SendMail = 1
