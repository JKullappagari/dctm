Asset Data Excel Report

 

Asset Data Excel report will generate excel file with all assets data and all possible data columns in DCTrackMobile. Users can filter or select columns that needs to be displayed in the excel file.

 

To view Asset Data Excel Transaction Report,

 

1. Navigate Reports>>Asset Data Excel Report in the DCTrackMobileWeb menu bar. 

You will see the Asset Data Excel Report screen.
