Delete Application Criticality

 

To delete an Application Criticality,

1.      Navigate Application>>Application Criticality on the DCTrackMobileWeb menu bar.

2.      On the Application Criticality screen, tick on the check box corresponding to the Application Criticality to be deleted from the list.

3.      Click Delete Selected button.

You will see the confirmation message window.

Figure 207: Application Criticality Screen – Delete Confirmation Message

Can I select multiple Application Criticalities for deletion?

Yes. Simply tick on the check box corresponding to the Criticality you would like to remove before clicking on the Delete Selected button. You can also click the Select All check box to select all items in the list for deletion.

 

4.      Click OK  button to proceed or click Cancel  button to disregard operation.

Upon deletion, you will see the successful deletion message and the deleted Application Criticality is no longer on the list.

Figure 208: Application Criticality Screen – Successful Deletion
