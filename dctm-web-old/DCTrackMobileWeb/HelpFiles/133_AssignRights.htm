Assign Rights

 

Assign Rights is used to assign access rights to a particular user. This feature will be used to add user to a group. So that user will inherit all the access rights of that group.

1.     Click on Assign Groups icon  correspond to a user, who needs to be added to a group in User Search screen.

2.     Check the group to which user need to be added under the groups available.

3.     Click Save button   to save the details.
