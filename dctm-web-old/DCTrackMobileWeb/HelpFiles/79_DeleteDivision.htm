Delete Division

 

To delete a Division,

1.      Navigate Application>>Division on the DCTrackMobileWeb menu bar.

2.      On the Division screen, tick on the check box corresponding to the Division to be deleted from the list.

3.      Click Delete Selected button.

You will see the confirmation message window.

Figure 101: Division Screen – Delete Confirmation Message

Can I select multiple Divisions for deletion?

Yes. Simply tick on the check box corresponding to the Divisions you would like to remove before clicking on the Delete Selected button. You can also click the Select All check box to select all items in the list for deletion.

 

4.      Click OK  button to proceed or click Cancel  button to disregard operation.

Upon deletion, you will see the successful deletion message and the deleted Division is no longer on the list.

Figure 102: Division Screen – Successful Deletion
