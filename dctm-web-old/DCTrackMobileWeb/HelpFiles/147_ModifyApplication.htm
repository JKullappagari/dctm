Modify Application

 

To modify an Application,

1.      Navigate Application>>Create Application on the DCTrackMobileWeb menu bar.

2.      On the Application screen, click the Edit  icon corresponding to the Application you want to modify.

You will see the existing values displayed on the screen.

Figure 198: Application Screen – Edit details

3.      Edit the details to be modified.

4.      Click Save  button to update the database with the changes or click Reset button to refresh page fields.

Upon saving, you will see the successful update message.
