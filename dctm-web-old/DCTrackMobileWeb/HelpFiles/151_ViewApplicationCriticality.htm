View Application Criticality

 

To view an Application Criticality,

1.      Navigate Application>>Application Criticality on the DCTrackMobileWeb menu bar.

You will see the Application Criticality Screen with a listing of all created Application Criticalities with corresponding descriptions.

Figure 204: Application Criticality Screen – View Site details

2.      Click page number hyperlink, if applicable to view the list in another page(s).
