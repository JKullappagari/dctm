App Summary - Rack Report

 

Application Summary - Rack report will show asset details including applications running in that asset. Application details can be color coded by application criticality. Application criticality color codes will be defined using “Application Criticality” page under “Application” menu. Report data can be filtered to show data of a particular site, application criticality or even a specific application.

 

To view App Summary - Rack Report,

 

1. Navigate Reports>>App Summary – Rack Report in the DCTrackMobileWeb menu bar. 

You will see the App Summary - Rack Report screen.
