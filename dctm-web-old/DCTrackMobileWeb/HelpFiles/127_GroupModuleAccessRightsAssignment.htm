Group Module Access Rights Assignment

 

This section describes about providing the access rights to a particular group in the DCTrackMobileWeb system.

To assign rights to a group

 

1.      Navigate User Administration>>Group Module Access Rights Assignment from the DCTrackMobileWeb menu bar.

You will see the Group Module Access Rights Assignment screen.

Figure 169: Group Module Access Rights Assignment Screen

Figure 170: Group Module Access Rights Assignment Screen –with Modules

 

2.      Select a Group from the Group field dropdown option.

3.      Click on Expand/Collapse button.

All the available modules will be available in a tree view structure.

 

4.      Expand respective modules and provide access rights like create, modify, delete, view etc.

Figure 171: Group Module Access Rights Assignment Screen-Access rights assignment

5.      Click Save  button to store access rights data into the database

Figure 172: Group Module Access Rights Assignment Screen-Successful assignment
