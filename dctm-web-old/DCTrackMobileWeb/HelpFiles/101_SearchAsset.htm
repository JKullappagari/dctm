Search Asset

 

Search Asset is the most important function of the system. Using this option, user can search the assets using various criteria and also perform different transactions for the asset such as – Decommissioning, Write off etc. This section describes about searching for an asset with the following search options:

·       Site

·       Location

·       Custodian

·       Asset Type

·       Tag Status (RFID assigned/ Not assigned)

·       Asset Status

o   Created/RFID Assigned/Barred/Restricted/Decommissioned/Write-Off

·       Host name

·       Asset Name

·       Tag No (EPC value) – Barcode or RFID value

·       Manufacturer

·       Asset Model

·       Serial No

1.      Navigate Asset>>Search Asset on the DCTrackMobileWeb menu bar.
