Change Password

 

This section describes about changing password for the user.

To Change password

 

1. Navigate User Administration>>Change Password from the DCTrackMobileWeb menu bar.

You will see the Change Password screen.

Figure 180: Change Password Screen

 

 

Is Password complexity defined?

Yes. Password must be at least 6 characters, no more than 10 characters, and must include at least one upper case letter, one lower case letter, one numeric digit and no spaces and no special characters allowed.

 

3.     Enter the information in the requested fields.

4.     Follow the password complexity guidelines to define new password.

3. Click Confirm  button to change the password or click Reset button to refresh page fields.
