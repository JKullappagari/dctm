Capacity Report

 

Capacity report will give capacity in terms of U-space, Power and Weight information for each rack which satisfies selected location criteria.

 

To view Capacity Report,

 

1. Navigate Reports>>Capacity Report in the DCTrackMobileWeb menu bar. 

You will see the Capacity Report screen.
