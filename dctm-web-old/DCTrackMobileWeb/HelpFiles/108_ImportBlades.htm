Import Blades

 

This section describes about how to import Blade assets from an excel spreadsheet.

 

To import Blades,

1.      Navigate Asset>>Import Blades on the DCTrackMobileWeb menu bar.

You will see the Import Blades Screen.

Figure 137: Import Blades Screen

2.      Enter Company to which Assets will be added in the required field and Click Upload File  button to select the Asset file (excel file).

3.      Multiple files can be selected using Upload File option. Use Add button to add multiple files, also individual files can be removed using cross (X) next to the file.

4.      After selecting asset file(s) click on Upload  button to parse the file.

Figure 138: Import Blades - Multiple File selection

5.      Upload option will upload and parse all the selected files and shows combined file statistics.

Figure 139: Import Blades Screen after parse

 

 

6.      Once file is successfully parsed it will show the statistics in terms of how many Asset, unique sites, rooms, rows and racks, manufacturers, models and asset types found in the excel file.

7.      Click Import to DB  button to import the data into the DCTrackMobileWeb database.

Figure 140: final Import Blades Screen

 

8.      On successful import it will show the statistics of how many blade assets imported successfully and how many failed. For the records that has one or more errors, the detail of the error is not shown in this page. Instead only a ‘Success/Failure’ status is shown under the Reason column of the grid. This data can be exported to excel using excel by using the standard export functionality. This will be useful to view the details of the error messages, correct the input records and then import them again.
