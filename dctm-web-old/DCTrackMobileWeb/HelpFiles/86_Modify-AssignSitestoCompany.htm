Modify- Assign Sites to Company

 

To modify a Company – Site Assignment,

1.      Navigate Assignment >> Assign Sites To Company on the DCTrackMobileWeb menu bar.

2.      The list of current Available and Assigned sites will be displayed.

3.      Edit the current assignment by using the control arrows to assign or de-assign a site. Refer to steps 3 to 5 in Creating Company-Site Assignment for details.

4.      Click Save  button to update the database with the changes or click Reset  button to refresh page fields.

Upon saving, you will see the successful assignment message.
