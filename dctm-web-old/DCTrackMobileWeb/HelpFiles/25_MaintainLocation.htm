Maintain Location

 

The Location screen will enable privileged users to create and maintain the Locations. In a Data Center environment, a location can be either a Room, Row or a Rack and their hierarchy is explained earlier.

 

Can I create a Location of the same name?

Yes, as long as the location is attached to different parents. For example, one can have same ROW called A but for two different Rooms – Room1 and Room2. Similarly, one can have two or more racks named A11 but they are associated to different rows.
