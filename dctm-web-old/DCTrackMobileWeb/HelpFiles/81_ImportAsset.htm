Import Asset

To import Asset models,

1.      Navigate Masters>>Import Asset Models on the DCTrackMobileWeb menu bar.

You will see the Import Asset Models Screen.

Figure 103: Import Asset Models Screen

2.      Click Upload File  button to select the Asset models file (excel file).

3.      After selecting asset file(s) click on Upload  button to parse the file.

Figure 104: Import Asset Models - File selection

4.      Upload option will upload and parse all the selected files and shows combined file statistics.

Figure 105: Import Asset Models Screen after parse

 

 

5.      Once file is successfully parsed it will show the statistics in terms of how many Asset models, number of unique manufacturers and number of model types found in the excel file.

6.      Click Import to DB  button to import the data into the DCTrackMobileWeb database.

Figure 106: final Import Asset Models Screen

 

7.      On successful import it will show the statistics of how many assets models imported successfully and how many failed. For the records that has one or more errors, the detail of the error is not shown in this page. Instead only a ‘Success/Failure’ status is shown under the Reason column of the grid. This data can be exported to excel using excel by using the standard export functionality. This will be useful to view the details of the error messages, correct the input records and then import them again.
