Delete Manufacturer

 

To delete a Manufacturer,

5.      Navigate Masters>>Manufacturer on the DCTrackMobileWeb menu bar.

6.      On the Manufacturer screen, tick on the check box corresponding to the Manufacturer to be deleted from the list.

7.      Click Delete Selected  button.

You will see the confirmation message window.

 

Figure 46: Manufacturer Screen – Delete Confirmation Message

Can I select multiple Manufacturers for deletion?

Yes. Simply tick on the check box corresponding to the Manufacturer you would like to remove before clicking on the Delete Selected button. You can also click the Select All check box to select all items in the list for deletion.

 

8.      Click OK  button to proceed or click Cancel  button to disregard operation.

Upon deletion, you will see the successful deletion message and the deleted Manufacturer is no longer on the list.

 

 

Figure 47: Manufacturer Screen – Successful Deletion
