Application

 

This chapter describes the Application module.  The system administrators and users are allowed to perform all admin activities like creating applications, other application properties like application type, application criticality, and division and also map application(s) to asset(s).

The following Site Access related functionalities can be accessed from the Application menu on the DCTrackMobileWeb navigation bar:

·       Application Map

·       Application Type

·       Create Application

·       Application Criticality

·       Division
