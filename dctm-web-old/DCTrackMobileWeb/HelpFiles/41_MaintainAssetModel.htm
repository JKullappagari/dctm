Maintain Asset Model

 

The Asset Model screen will enable privileged users to define and maintain the Asset Models list used by DCTrackMobile application. The models are maintained by manufacturer and hence before creating a model, one has to create the corresponding manufacturer.

 

Can I create an Asset Model of the same name?

No. Asset Model should be unique. System will inform you that the record already exists.

What is the purpose of IsBlade and IsEnclosure option?

IsBlade option will be used to identify blade or module asset models which will be placed inside an enclosure or chassis instead of rack. For all blade or module models, IsBlade option must be checked.   

IsEnclosure will be used to identify enclosure models like server enclosure, storage enclosure and network enclosure. Example: HP C7000                                      

 

In order to create an Asset Model, user has to specify below list of details:

 

Column Name

Description

ModelName

Model of the asset.  Alphanumeric, hyphen, underscore, dot, colon and single space only with maximum of 150 chars.

Manufacturer

Manufacturer of the asset.
Alphanumeric, hyphen, underscore, dot, colon and single space only with maximum of 50 chars.

Height (mm)

Height in mm

Width (mm)

Width in mm

Depth (mm)

Depth in mm

UHeight

U-height of the model in Rack Units (RU)

Weight (kg)

Weight in kg

Max Power in Watts

Maximum name plate power reading in Watts

Asset Type

Model Type like Server, Storage, Network, etc.

Total PSU Count

Total number of Power Supply Units

Required PSU Count

Minimum required Power Supply units. This is to calculate redundancy.

Power Connector Type PDUSide

Power cable connector type on PDU side like C13 or C19

Power Connector Type DeviceSide

Power Cable connector type on Device side like C14, C20, C15, etc.

IsBlade

True / False indicating whether model is a blade or Module

IsEnclosure

True/ False indicating whether model is Enclosure

MountType

RackMount, Standalone, Stack in rack, vertical mount, enclosure mount.

AirFlowDirection

Front to Rear or Rear to Front. This will be helpful for mounting asset into the rack

InternalWith_Rack

Applicable to Racks only, internal width of a rack.

Internal_Depth_Rack

Applicable to racks only. Internal depth of a rack

Enclosure Front/Rear – Row and Column count

Applicable to Enclosure models only. Row X Column count for front and rear orientation

Blade – Row X Column count

Applicable to blade models only. Row X Column count indicates the size of blade like whether it’s a full height or half-height, etc.
