Modify Technology Category

 

To modify a Technology Category,

1.      NavigateMasters>>Technology Category on the DCTrackMobileWeb menu bar.

2.      On the Technology Category screen, click the Edit  icon corresponding to the Technology Category you want to modify.

You will see the existing values displayed on the screen.

Figure 56: Technology Category Screen – Edit details

3.      Edit the details to be modified.

4.      Click Save  button to update the database with the changes or click Reset button to refresh page fields.

Upon saving, you will see the successful update message.

 

Figure 57: Technology Category Screen – Successful Update
