Asset Transaction Report

 

Asset Transaction report will show asset transaction details within a specified date range.  Report data can be filtered to show transactions of a particular asset, transactions by a specific location and also list based on particular transaction type.

 

To view Asset Transaction Report,

 

1. Navigate Reports>>Asset Transaction Report in the DCTrackMobileWeb menu bar. 

You will see the Asset Transaction Report screen.
